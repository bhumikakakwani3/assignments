programming languages

A programming language is a system of noatation for writing computer programs. programming languags are described in terms of their syntax (form) and semantics (meaning), usually defined by a formal language. languages usually provide features such as a type system, variables and mechanism for error handling. an implementation of a programming language is required in order to execute programs, namely an interpreter or a compliar. an interpreter directly executes the source code, while a compliarproduces an executable program.
